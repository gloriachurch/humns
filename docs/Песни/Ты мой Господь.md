# Ты мой Господь  
Ты мой Господь  
Ты моя любовь  
Все что есть во мне  
Принадлежит Тебе  
  
Вся жизнь моя  
Наполнена Тобой  
Только на Тебя  
Надеюсь я Бог мой  
  
Ты жизнь свою отдал  
Грехи мои забрал  
Сейчас  Тебе хочу  
Вознести хвалу  
  
Тебе вся слава  
Тебе вся честь  
Ты Бог всех Богов  
Ты Царь всех царей  
Тебе вся слава  
Тебе вся честь  
Спаситель  ты мой  
Всегда будь со мной
