# Ты - мира свет, воссиявший во тьме
Ты - мира свет, воссиявший во тьме,  
Дай мне увидеть Тебя!  
Сердце поёт о Твоей красоте:  
Жаждет с Тобой быть всегда.  
  
Я здесь, чтобы славить, я здесь, чтоб склониться;  
Я здесь, чтоб сказать, что Ты - мой Бог!  
Нет Тебя прекрасней, нет Тебя достойней,  
Нет Тебя чудесней для меня!  
  
Царь всех времён, высоко вознесённый,  
Славный на небесах,  
Смиренно пришёл в мир, Тобой сотворённый,  
Отдал Свою жизнь за всех нас!
