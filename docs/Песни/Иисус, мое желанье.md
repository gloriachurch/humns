# Иисус, мое желанье
Иисус, моё желанье -  
Воздавать Тебе хвалу!  
Господь, всем моим сердцем Тебе поклонюсь,  
Признаюсь Тебе в любви моей,  
Всё, чем восхищаюсь - только в Тебе.  
  
Сердце отдаю Тебе и душу мою,  
Лишь для Тебя живу!  
В каждом шаге моём  
Быть хочу с Тобой вдвоём,  
Да будет Твоя воля во всём!
