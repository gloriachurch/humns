# Вселенная вся
Вселенная вся превозносит Царя.  
Чудеса и слава Его по всей земле!  
Небеса воспоют, горы провозгласят,  
Всё творенье восстанет в хвале  
Пред Царём царей!  
  
Тебе поклоняюсь,  
Тебе поклоняюсь,  
Иисус Христос:  
Он - Господь!  
Познай, вся земля,  
Склонись, вся земля,  
Иисус Христос:  
Он - Господь!  
  
Вечный Господь,  
Грядуший опять,  
Всё в мире пройдёт,  
Только правда Твоя  
Будет вечно стоять!  
  
Аллилуйя, Аллилуйя,  
Ты достоин чести, хвалы!
