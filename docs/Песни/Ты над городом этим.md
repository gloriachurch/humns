# Ты над городом этим
Ты над городом этим  
И над этим народом,  
И над этой страною,  
Господь!  
  
Ты во тьме освещаешь,  
Озаряешь надеждой,  
Утешаешь Собою,  
Господь!  
  
Нет Такого, как наш Бог!  
Нет Такого, как наш Бог!  
  
Грядет пора больших чудес,  
Господь прольет благодать с небес в этот город!
